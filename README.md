# Cuenta BPM
## _Open Source BPM counter_


[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://google.com)

Cuenta BPM is a counter for BPM. It helps DJs to count BPM easily.

## Features

- Receive Taps
- Show BPM
- Restart counter
